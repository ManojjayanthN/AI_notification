# AI-Powered Notification System

## Overview
This project is an AI-powered notification and reminder system for schools and colleges.  
It allows staff to send automated notifications to students about exams, deadlines, or events and track delivery and engagement.  

**Features:**
- Student portal to view notifications
- Scheduler sends notifications automatically
- AI-powered analysis for engagement optimization
- Logs delivery status in both SQLite database and CSV file
